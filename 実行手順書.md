# キーワード分析レポート生成ツール - 実行手順書

## 概要
StanbyADのキーワードデータから分析レポートを生成するツールです。
**アカウント分析くん**が自動的にファイルを検出・整理して分析を実行します。

## データの取得元

### 1. キーワードレポートのローデータ
**databricksから取得**
- アカウントIDと期間を指定して取得
- 取得したデータをExcelファイルとして保存
- ファイル名に「キーワード別レポート」を含む必要があります

### 2. キャンペーン別の合計データ
**シャーレのキャンペーンレポートから取得**
- キャンペーンレポートからCSVをダウンロード
- ファイル名に日付範囲を含む必要があります（例: `20260101-20260131`）
- ファイル名に`Campaigns`を含む必要があります
- 前期間と後期間の2つ以上のファイルが必要です

## セットアップ（初回のみ・必須）

### ステップ1: Pythonのインストール（必須）
このツールを実行するためには、**Python（パイソン）**というプログラム言語の実行環境が必要です。

#### Pythonとは
- データ分析などで使われるプログラム言語です
- このツールを動かすために必要な基盤ソフトウェアです

#### インストール方法

**macOSの場合：**
1. ブラウザで以下のURLを開く：
   https://www.python.org/downloads/
2. 「Download Python 3.x.x」（最新版）ボタンをクリック
3. ダウンロードしたファイル（例: `python-3.x.x-macos.pkg`）をダブルクリック
4. インストーラーの指示に従って進める
   - 「続ける」をクリック
   - 「インストール」をクリック
   - パスワードを入力（求められた場合）
5. インストールが完了したら、以下の手順で確認：
   - ターミナル（アプリケーション > ユーティリティ > ターミナル）を開く
   - 以下のコマンドを入力してEnterキーを押す：
     ```bash
     python3 --version
     ```
   - `Python 3.x.x`のようにバージョンが表示されればOK

**Windowsの場合：**
1. ブラウザで以下のURLを開く：
   https://www.python.org/downloads/
2. 「Download Python 3.x.x」（最新版）ボタンをクリック
3. ダウンロードしたファイル（例: `python-3.x.x-amd64.exe`）をダブルクリック
4. インストーラーで**必ず「Add Python to PATH」にチェックを入れる**
5. 「Install Now」をクリック
6. インストールが完了したら、以下の手順で確認：
   - コマンドプロンプト（Windowsキー + R → `cmd`と入力してEnter）を開く
   - 以下のコマンドを入力してEnterキーを押す：
     ```bash
     python --version
     ```
   - `Python 3.x.x`のようにバージョンが表示されればOK

**注意事項：**
- Python 3.9以上が必要です（3.9, 3.10, 3.11, 3.12など）
- すでにインストールされている場合でも、上記の確認コマンドでバージョンを確認してください
- インストールに時間がかかる場合があります（5〜10分程度）

### ステップ2: ツールのダウンロードと展開
1. このツールのフォルダを任意の場所に配置してください（例: デスクトップ）
2. フォルダ名は`keywordanalysis`のままでOKです

### ステップ3: 初回実行時の自動セットアップ
1. **macOSの場合**: `アカウント分析くん.command`をダブルクリック
2. **Windowsの場合**: ターミナル（コマンドプロンプト）で以下のコマンドを実行：
   ```bash
   cd [ツールのフォルダのパス]
   python account_analyzer.py
   ```
3. 初回実行時は、必要なライブラリ（追加のプログラム）が自動でインストールされます
   - インターネット接続が必要です
   - 数分かかる場合があります
4. 2回目以降は通常の実行のみです（自動セットアップはスキップされます）

## 実行手順（2回目以降）

### ステップ1: データファイルのダウンロード
必要なデータファイルをダウンロードフォルダに保存してください。

1. **キーワードレポートのローデータ**
   - databricksから取得したExcelファイルをダウンロードフォルダに保存
   - ファイル名に「キーワード別レポート」を含めてください

2. **キャンペーン別の合計データ**
   - シャーレからダウンロードしたCampaigns.csvファイルをダウンロードフォルダに保存
   - 前期間と後期間の2つ以上のファイルが必要です
   - ファイル名には日付範囲（例: `20260101-20260131`）と`Campaigns`を含めてください

**重要：** ファイルは**ダウンロードフォルダに保存するだけ**でOKです。手動でフォルダに移動する必要はありません。

### ステップ2: アカウント分析くんの実行

#### macOSの場合
1. `アカウント分析くん.command`をダブルクリック
2. ターミナルが開いて処理が開始されます

#### Windowsの場合（またはmacOSでターミナルから実行）
1. ターミナル（コマンドプロンプト）を開く
2. プロジェクトディレクトリに移動：
   ```bash
   cd [ツールのフォルダのパス]
   ```
   例: `cd C:\Users\あなたの名前\Desktop\keywordanalysis`
3. 以下のコマンドを実行：
   ```bash
   python account_analyzer.py
   ```

### ステップ3: ファイルの選択と整理

アカウント分析くんが自動的にダウンロードフォルダからファイルを検出します。

#### 3-1. 比較したいキャンペーンレポートを選択
- 検出されたCampaigns.csvファイルの一覧が表示されます
- 各ファイルには番号が振られています
- **2つ以上のファイルを選択**してください（後期間と前期間のデータが必要です）
- 選択方法：
  - 個別選択：`1,3,5`のようにカンマ区切りで番号を入力
  - 範囲指定：`1-5`のように範囲で指定
- 選択後、自動的に最新のファイルが後期間、古いファイルが前期間として分類されます

#### 3-2. キーワード別レポートファイルの選択
- 検出されたキーワード別レポートファイルの一覧が表示されます
- 使用するファイルの番号を入力してください
- 選択方法：
  - 個別選択：`1,3,5`のようにカンマ区切りで番号を入力
  - 範囲指定：`1-10`のように範囲で指定
  - スキップ：Enterキーを押すと既存のファイルがあれば使用します
- 選択後、後期間と前期間への分類方法を選択します：
  1. **自動分類（日付で判定）**：ファイル名の日付から自動的に分類
  2. **手動で指定**：後期間に入れるファイルの番号を指定
  3. **すべて後期間に入れる**：選択したファイルをすべて後期間に配置
  4. **すべて前期間に入れる**：選択したファイルをすべて前期間に配置

#### 3-3. ファイルの自動移動
- 選択したファイルは自動的に適切なフォルダに移動（コピー）されます：
  - Campaigns.csv → `data/totals/`
  - キーワード別レポート → `data/後期間/` または `data/前期間/`
- 元のファイルはダウンロードフォルダに残ります

### ステップ4: ファイル準備状況の確認
- 必要なファイルが揃っているか確認画面が表示されます
- 不足しているファイルがある場合は警告が表示されますが、続行することもできます

### ステップ5: キャンペーン合算機能の設定
- 以下のメッセージが表示されます：
  ```
  === キャンペーン合算機能の設定 ===
  合算するキーワード条件を入力してください（カンマ区切り、例: 看護師,介護）:
  ```
- **合算機能を使う場合**：
  - 合算したいキャンペーン名に含まれるキーワードをカンマ区切りで入力してください
  - 例: `看護師,介護` と入力すると、「看護師」を含むキャンペーンと「介護」を含むキャンペーンがそれぞれ別のグループとして合算されます
  - 入力後、Enterキーを押してください
- **合算機能を使わない場合**：
  - 何も入力せずにEnterキーを押してください
  - 合算機能は無効化され、通常の分析が実行されます

#### 合算を指定した場合のサマリー出力について（重要）
- **合算を指定した場合、サマリーの「キャンペーン別の指標」は合算後キャンペーン単位で前後比較**されます
  - Campaigns.csvの合計値も同じルールで合算して比較します

### ステップ6: 分析の実行
- 設定が完了すると、自動的に分析が開始されます
- 処理が完了するまで待ちます（数分かかる場合があります）

### ステップ7: 結果の確認
- `output/`ディレクトリにExcelファイルが生成されます
- ファイル名: `[アカウント名]_KWレポート.xlsx`
- 生成されたファイルを開いて内容を確認してください

## 注意（`run.py` について）
- `run.py` は**キャンペーン合算に非対応**です（合算ONで実行すると `_2` シート複製やCV/平均CPAの不整合が起きるため、現在はエラー停止します）。\n+  合算する場合は `アカウント分析くん.command` もしくは `python run_grouping.py` を使用してください。

## 出力レポートの内容

生成されるExcelファイルには以下のシートが含まれます：

1. **サマリーシート**
   - アカウント全体の指標
   - キャンペーン別の指標とシェアの変化
   - 影響の大きいキーワードTOP5

2. **アカウントシート**（前期間・後期間別）
   - COST降順、CV降順、CPA昇順で3パターン表示
   - CPA列に条件付き書式（平均からの乖離を色分け）

3. **キャンペーンシート**（各キャンペーンごと、前期間・後期間別）
   - アカウントシートと同様の構成
   - キャンペーン単位のCPA平均で条件付き書式を適用

## トラブルシューティング

### Pythonがインストールされているか確認できない
- ターミナル（コマンドプロンプト）で`python3 --version`（macOS）または`python --version`（Windows）を実行
- バージョンが表示されない場合は、Pythonのインストールが完了していません
- 上記の「ステップ1: Pythonのインストール」を再度確認してください

### ライブラリのインストールに失敗する
- インターネット接続を確認してください
- ファイアウォールやセキュリティソフトがブロックしていないか確認してください
- それでも解決しない場合は、以下のコマンドを手動で実行してみてください：
  ```bash
  pip3 install -r requirements.txt
  ```
  （Windowsの場合は`pip install -r requirements.txt`）

### ファイルが検出されない
- ダウンロードフォルダにファイルが保存されているか確認してください
- ファイル名が正しいか確認してください：
  - Campaigns.csvファイル：ファイル名に`Campaigns`と日付範囲（例: `20260101-20260131`）を含める
  - キーワード別レポート：ファイル名に「キーワード別レポート」を含める
- 既存のファイルがある場合は、それを使用して分析を続行できます

### ファイルの選択でエラーが発生する
- 番号の入力形式を確認してください（例: `1,3,5` または `1-5`）
- Campaigns.csvファイルは2つ以上選択する必要があります
- 範囲指定の場合は `開始番号-終了番号` の形式で入力してください

### エラーが発生した場合
エラーが発生した場合は、ターミナル（コマンドプロンプト）に表示されたエラーメッセージをそのままコピーして、武田にSlackで連絡してください。
